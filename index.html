PROJECTS

  • WakeApp

    PlaystoreGithub

    WakeApp is an Android native mobile app designed for transit commuters who enjoy sleeping on the bus. Simply enter in your destination, and the app will track the distance between you and your destination. As you approach your stop, the app will notify you that your destination is near, ensuring you do not miss your stop. WakeApp has the ability to handle trip transfers, as well as place autocomplete. Check it out in the Google Playstore and never miss your stop again.


    WakeApp was built in Android Studio with Java.

  • Honey Trail

    PlaystoreGithub

    Honey Trail is an tap mobile game built in Unity and written in C#. The goal of the game is to collect as many honey drops as possible before they exit the screen. Honey Trail is available in the Google Play store - download it and see if you can get the highscore!


    Honey Trail was built in Unity with C#.

  • Canary

    Github

    Canary provides a framework to help monitor conditions of miners to ensure their safety through an Android smartwatch. Miners will be wearing an Android smartwatch that measures and streams data to a central server via MQTT protocol. The data is then analyzed for sudden increase of acceleration or prolonged idling, which indicates potentially dangerous situations. This data is displayed on a web app, which includes acceleration graphs as well as acceleration live streams.


    Team project - front-end UI built using AngularJS and Bootstrap.

  • Kontribute

    Github

    Kontribute is a hybrid mobile app that allows for easy and efficent event planning. Each event has a Kontribution list which notifies what the event participants should bring. This removes the stress of having to manually allocate what items participants need to bring. For instance, a Kontribution list could include snacks, utensils, game controllers, etc. The participants will then put their name under the list items that they are able to bring.


    Team project - front-end UI navigation features, login logic, and database configuration implemented using AngularJS, Ionic, and Firebase.

  • Freezeframe

    Github

    Freezeframe is a puzzle/precision platformer where you control a square. The key game mechanic is the square's ability to clone itself in order to use the clone as a platform or stepping stone to get to certain areas of a level.


    Team project - game portal mechanics and level design. Built in Unity with C#.
